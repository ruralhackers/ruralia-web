## 1. Resumen ejecutivo

Rural IA opera en un nicho con **competencia SEO nula en keywords exactas** y **demanda creciente en keywords madre**. Las busquedas relacionadas con "inteligencia artificial" han crecido +600% segun Google Trends.

**Hallazgo clave (validado con Ahrefs):**
- **Nadie posiciona** para keywords de nicho (retiro IA, curso IA presencial, IA freelancers). Son keywords sin volumen significativo (<10/mes) y sin competidores indexados.
- **El volumen real esta en keywords genericas:** "curso inteligencia artificial" (1900/mes, KD 40), "herramientas ia" (400/mes, KD 52, TP 1500), "bootcamp ia" (200/mes, KD 46).
- **El long-tail es muy escaso:** Solo 13 keywords derivadas con KD<30, la mejor con 50 busquedas/mes.

**Implicacion estrategica:** SEO no sera el canal principal de captacion de clientes a corto plazo. Con un dominio nuevo y keywords de nicho sin volumen, el trafico organico directo sera limitado los primeros 3-6 meses. **El SEO debe ser un juego de medio-largo plazo** centrado en:
1. **Blog informacional** atacando keywords con volumen real (herramientas IA, automatizacion, comparativas) para generar trafico top-of-funnel
2. **Autoridad tematica** para eventualmente posicionar en "curso inteligencia artificial" (1900/mes)
3. **Rich snippets** (FAQPage schema) para maximizar CTR en las pocas impresiones iniciales
4. **SEO local** (Google Business Profile) como canal complementario

**Canales de captacion prioritarios (corto plazo):** Redes sociales, comunidad Rural Hackers, ads pagados, email marketing. SEO crece en paralelo.

### 3.2 Interlinking strategy

Cada post del blog debe enlazar a:
1. La pillar page `/retiro-ia-galicia/` (CTA final siempre)

3. A `/para-quien/` cuando sea relevante

La pagina de conversion `/retiro-ia-galicia/` debe recibir enlaces internos desde **todas** las demas paginas.

## 5. SEO tecnico

### 5.1 Checklist tecnico prioritario

- [x] Framework SEO-friendly (Astro SSG = HTML estatico, perfecto)
- [x] Sitemap XML automatico (@astrojs/sitemap)
- [x] Componente SEO reutilizable (ya existe)
- [x] URLs limpias y descriptivas (ya planificadas)
- [x] Idioma declarado (`lang="es"`)
- [ ] **robots.txt** — Crear en `/public/robots.txt`
- [ ] **Canonical tags** — Verificar en cada pagina
- [ ] **Open Graph images** — Crear OG image por pagina (1200x630)
- [ ] **Favicon completo** — Agregar apple-touch-icon, manifest
- [ ] **Hreflang** — No necesario ahora (solo espanol)
- [ ] **Google Search Console** — Verificar propiedad del dominio
- [ ] **Structured data testing** — Validar con Rich Results Test

### 5.2 robots.txt recomendado

```
User-agent: *
Allow: /
Disallow: /api/
Disallow: /admin/

Sitemap: https://ruralia.gal/sitemap-index.xml
```

### 5.3 Schema markup por pagina

| Pagina | Schema types |
|--------|-------------|
| Home | `Organization` + `WebSite` + `Event` |
| Retiro IA Galicia | `Event` + `Offer` + `Place` |

| Blog posts | `Article` + `BreadcrumbList` |
| Sobre nosotros | `Organization` + `Person` (equipo) |

### 5.4 Core Web Vitals (ventaja de Astro)

Astro genera HTML estatico con zero JS por defecto. Esto da ventaja enorme:

| Metrica | Target | Expectativa con Astro |
|---------|--------|-----------------------|
| LCP | < 2.5s | < 1.5s (excelente) |
| FID/INP | < 100ms | < 50ms (excelente) |
| CLS | < 0.1 | ~0 (excelente) |

**Acciones para mantener rendimiento:**
- Servir imagenes en WebP con dimensiones explicitas
- Lazy loading para imagenes below the fold
- Preload de la fuente principal si se usa custom font
- Minimo JavaScript (solo lo esencial: menu mobile, analytics)

- Astro SSG garantiza rendimiento tecnico excelente (Lighthouse 100)
- Nicho sin competencia directa = posicionamiento rapido
- Dominio ruralia.gal es memorable y brandeable
- Contenido rico y diferenciado (experiencia unica)

### Riesgos SEO a mitigar
- Dominio nuevo = sin autoridad. Mitigar con backlinks de partners desde dia 1
- Volumen de busqueda limitado en keywords exactas. Mitigar con long-tail + blog informacional
- Estacionalidad (ediciones mayo-diciembre). Mitigar con contenido evergreen que genere trafico todo el ano

### Proyeccion realista
- **Mes 1-2:** Indexacion, primeras impresiones, 0-100 clicks organicos
- **Mes 3-4:** Primeras posiciones top 10, 200-500 clicks/mes
- **Mes 6:** Autoridad creciente, 800-1500 clicks/mes
- **Mes 12:** Canal organico consolidado, 2000-5000 clicks/mes
